# J programming language Memo
```
2 (3!:4) y	数値yを32bitビッグエンディアンとしてASCII列化する
1 (3!:4) y	数値yを16bitビッグエンディアンとしてASCII列化する
1 ic y		同上
x,y			文字列xとyを結合したものを返す
x 1!:2<F	ファイル名Fに文字列xを書き込む
i.n			0から数値n未満までの整数列を生成する
x $ n		数値nをx個もつ配列を生成
y x $ n		数値nをy行x列に並べた配列を生成
x % y		xをyで割った値を返す
x | y		yをxで割ったあまりを返す
p =: q		pにqを代入
x {a.		数値または数値を要素とする配列xを対応するASCII文字に変換
m}T			配列Tからベクトルmに従って値を取り出す　ex: (3|i.4)}(3 4 $i.12)
<.x			floor(x)
#:x			xを2進数に変換して各桁の配列を返す．
```
